# Flipkart Sales Analysis

## Description
This project is about analyzing Flipkart sales data using Python.  
I used data analysis and visualization to understand sales trends, product performance, and customer behavior.

## Tools Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## What I did in this project
- Cleaned the dataset  
- Analyzed sales and product data  
- Created visualizations for better understanding  
- Found insights about top products and categories  

## How to run this project
1. Clone the repository  
2. Open the notebook file  
3. Install required libraries  
4. Run all cells
